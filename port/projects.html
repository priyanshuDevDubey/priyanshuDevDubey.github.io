Wordle

A simple word guessing game called 'wordle', which is made using html, css, and javascript , in this project the grid and boxes are implemented using js and html gives structure while css gives some design.

This game comprises a 6*5 grid structure made of boxes and the algorithm picks a random 5 letter word from dictioniry attached, and the user has to guess the word in six attempt, each letter user enters is read by keylistener() function and is matched and if the word fully matches then it shows congratulations message.

Energy Estimation

It is an analysis of world and Indian solar energy Production , It studies the trend in the production of solar energy in different countries of the world and a seperate analysis for india's solar energy production .

The model works on the data which used to train the algorithm to plots regression curves (linear and polynomial) , then based on the data , the model produces an estimation of solar energy production in world (and India seperately) , the correlation coefficient is also calculated for the model to judge the correctness of model.
